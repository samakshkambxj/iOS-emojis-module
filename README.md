# iOS-emojis-module
A systemless module to replace stock android emojis with iOS 18.4 emojis

## Installation
1. Download the latest release from the [Releases page](https://github.com/samakshkambxj/iOS-emojis-module/releases/latest).
2. Open Magisk/KernelSU/APatch.
3. Go to **Modules** â†’ **Install** and select the downloaded ZIP file.
4. Reboot your device.
5. Enjoy iOS emojis system-wide!

6. ## Compatibility
- **Magisk Version**: Requires Magisk v24.0 or higher.
- **Android Version**: Tested on Android 10-16. May work on older versions, but not guaranteed.
- **Devices**: Works on most devices.
